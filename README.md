# 낯선 언어와 프레임워크를 사용해 데스크탑 애플리케이션 개발

## 날씨찾기 데스크탑 애플리케이션 개발
---
### 프로젝트 요구사항

* IDE IntelliJ IDEA Community 사용
* JDK Java 21 사용
* 빌드 도구 Gradle (Kotlin DSL)
* Kotlin 1.9 이상
* UI 프레임워크 Jetpack Compose for Desktop 사용
* 외부 시스템(api)  기상청_단기예보 ((구)_동네예보) 조회서비스(https://www.data.go.kr/index.do)
---
### 기능 요구사항

* GUI를 통해 지역을 선택할 수 있다..
* 해당 지역의 하루동안의 날씨(온도, 습도, 미세먼지)를 시간대별로 확인할 수 있어야한다.


## 세부 기능 요구사항
* 지역을 선택할수 있다.
  - 존재하지 않는 지역을 검색할시 알림창을 보여준다.

**지역목록 :** 

| 서울 | 인천 | 부산 | 대전 |
|------|------|------|------|
| 대구 | 광주 | 울산 | 세종 |
| 경기 | 강원 | 충남 | 충북 |
| 전남 | 전북 | 경남 | 경북 |
| 제주 |  |  |  |

    
* 선택한 지역의 온도, 습도, 미세먼지, 강수량을 볼 수 있다.
* 뒤로가기 버튼을 통해 메인화면(지역을 선택하는 화면)으로 이동할 수 있다.
